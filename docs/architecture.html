Tech Stack

Frontend Client

  • Vue 3.4 with Composition API
  • Vue Router 4.3 (7 routes)
  • Axios 1.6 HTTP client
  • Vite 5.2 dev server & bundler
  • Custom SVG charts (no chart library)
  • i18n: English & Japanese

Backend Server

  • Python 3.11+ with FastAPI
  • Uvicorn ASGI server
  • Pydantic v2 validation
  • CORS middleware (all origins)
  • 17 REST API endpoints
  • No database — in-memory only

Data Storage

  • 7 JSON files in server/data/
  • Loaded once at server startup
  • No persistence (restart resets)
  • Categories: Circuit Boards, Sensors, Actuators, Controllers, Power Supplies
  • Warehouses: San Francisco, London, Tokyo

Testing QA

  • pytest with FastAPI TestClient
  • pytest-asyncio for async tests
  • httpx for HTTP testing
  • pytest-cov for coverage
  • Playwright for browser testing

Frontend Architecture

Composables (Shared State)

  • useFilters() — singleton filter state shared across all views
  • useAuth() — current user profile, locale-aware
  • useI18n() — translations (EN/JA), currency formatting (USD/JPY)

Components (9 total)

  • FilterBar — 4-filter toolbar on every page
  • Detail Modals — Inventory, Order, Product, Backlog, Cost
  • TasksModal — Task create/toggle/delete
  • ProfileMenu + LanguageSwitcher

Key Patterns

  • Composition API exclusively (no Options API)
  • Reactive refs for state, computed for derived data
  • Props down, events up for component communication
  • Centralized API calls in api.js via Axios
  • Scoped styles per component
